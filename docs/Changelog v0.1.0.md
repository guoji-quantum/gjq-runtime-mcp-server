## [0.1.0] - 2026-07-06

#### 🎉 首次发布

#### ✨ 核心特性
- 提供完整 MCP 工具集：账号、设备、计算任务、任务状态/结果/日志/详情。
- 支持 OpenQASM 2.0，兼容 OpenQASM 3（需安装 `qiskit_qasm3_import`）。
- 提供示例电路 MCP 资源：`bell-state` / `ghz-state` / `superposition` / `random`。

## [0.1.0] - 2026-07-06

#### 🎉 Initial Release

#### ✨ Features
- Full MCP toolset for account, device, compute submission, and task status/result/log/detail workflows.
- OpenQASM 2.0 support out of the box; OpenQASM 3 supported with optional `qiskit_qasm3_import`.
- Example circuit resources: `bell-state`, `ghz-state`, `superposition`, `random`.
